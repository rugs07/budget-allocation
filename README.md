# Budget Allocation Popup (React)

This is a Vite + React project for the Namastedev Practice Platform problem: Budget Allocation Popup.

## How to use

1. Install dependencies:
   ```
   npm install
   ```
2. Start the dev server:
   ```
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Switch between Solution and Template
- To use the starter template, open `src/App.jsx` and comment/uncomment the import lines for `BudgetPopup` and `BudgetPopupTemplate` as needed.

## Files
- `src/BudgetPopup.jsx`: Solution
- `src/BudgetPopupTemplate.jsx`: Starter template
- `description.md`, `hints.md`, `testcases.md`, etc.: Problem metadata
